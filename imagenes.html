¿Cómo se pone una imagen?

<img src="images/gato.jpg" alt="Un gatito">
  • src="images/gato.jpg": Es la ruta donde se encuentra la imagen, en este caso dentro de la carpeta del proyecto tengo una que se llama images, ahí dentro está la foto.
  • alt="Un gatito": La etiqueta alt sirve para mostrar el texto si la imagen no carga o la ruta es incorrecta y también para que los lectores de pantalla puedan leer el texto.
