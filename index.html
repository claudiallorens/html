HTML

¿Qué es?

HTML significa HyperText Markup Language (Lenguaje de Marcado de Hipertexto). Es el lenguaje básico que se usa para crear páginas web y describir su estructura. Fue creado en 1991 por Tim Berners-Lee, un científico británico, con el objetivo de facilitar la creación y visualización de documentos en la World Wide Web (WWW).

¿Para qué se usa?

HTML se usa para estructurar una página web. Define cosas como:

  • Encabezados
  • Textos
  • Listas
  • Enlaces
  • Imágenes

¿Qué se suele usar junto a HTML?

  • CSS: Se usa para darle diseño y estilo a las páginas web, como colores, tamaños, márgenes, etc.
  • JavaScript: Es un lenguaje de programación que se usa principalmente para hacer que las páginas web sean interactivas (formularios, animaciones...).

Esta es la estructura básica de un archivo HTML:


    <!DOCTYPE html>
    <html lang="es">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Título de la página</title>
    </head>
    <body>
    
        <h1>Hola mundo</h1>
        <p>Lorem ipsum dolor sit amet consectetur adipisicing elit.</p>
    
    </body>
    </html>
  • <!DOCTYPE html>: Define la versión HTML.
  • <html>: Define el principio y el final del documento.
  • <head>: Contiene información como metadata, links a hojas de estilo y scripts.
  • <title>: Título de la web, aparece en la pestaña del buscador.
  • <body>: Aquí se incluye el contenido visible de la página (texto, imágenes y otros elementos).
  • <h1 - h6>: Títulos con diferente importancia.
  • <p>: Párrafo.
