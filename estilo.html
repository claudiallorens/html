Estilo con CSS

¿Qué es CSS?

CSS significa Cascading Style Sheets (Hojas de estilo en cascada).

Sirve para dar estilo a páginas HTML, como cambiar colores, tamaños, márgenes, posiciones, etc.

¿Cómo se puede usar CSS?

1. CSS externo (el más recomendado)

En el header del HTML: <link rel="stylesheet" href="style.css"> y crear archivo style.css

2. CSS en la cabecera

<style>
body { background-color: beige; }
</style>

3. CSS en línea (no recomendado)

<p style="color: red;">Texto en rojo</p>

Sintaxis básica

selector {
  propiedad: valor;
}

Ejemplo:

p {
  color: blue;
  font-size: 18px;
}

Selectores más usados

  • p: Selecciona todos los párrafos
  • .nombre-clase: Selecciona todos los elementos con esa clase
  • #nombre-id: Selecciona un único elemento con ese ID
  • *: Selecciona todos los elementos

Propiedades importantes

Estilo de texto:

  • color: cambia el color del texto
  • font-size: cambia el tamaño de la fuente
  • font-family: cambia el tipo de letra
  • text-align: alinea el texto
  • text-decoration: subraya, tacha, etc.

Estilo de cajas:

  • background-color: cambia el color de fondo
  • margin: espacio exterior
  • padding: espacio interior
  • border: añade un borde
  • width / height: tamaño del elemento

Colores

Se pueden escribir de distintas formas:

color: red;
color: #ff0000;
color: rgb(255, 0, 0);

Comentarios

body

body {

background-color: antiquewhite;

margin: 0px;

font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif;

}

El estilo aplicado al body define el fondo de la página con un color beige claro antiquewhite, quita los márgenes por defecto con margin: 0px, y establece una fuente sans-serif para todo el texto, utilizando una fuente más moderna y legible como 'Segoe UI'.

h1

h1 {

color: rgb(239, 91, 91);

text-align: center;

background-color: rgb(255, 202, 181);

width: 100%;

margin-top: 0px;

align-items: center;

padding: 20px 0;

}

El estilo de h1 asigna un color rojo claro al texto (rgb(239, 91, 91)) y un fondo beige claro (rgb(255, 202, 181)). Además, el texto se centra con text-align: center, y se le agrega un relleno en la parte superior e inferior de 20px.

h3

h3 {

color: rgb(239, 91, 91);

width: 100%;

align-items: center;

padding: 20px 0;

}

Al igual que en el estilo de h1, se aplica el mismo color al texto rgb(239, 91, 91), y se centra con padding: 20px 0 para agregar espacio superior e inferior.

nav y .navegacion

nav {

text-align: center;

width: 100%;

margin-top: -8px;

align-items: center;

padding: 10px 0;

}

.navegacion {

display: flex;

flex-direction: row;

list-style: none;

justify-content: space-around;

}

En el estilo de nav, se centra el contenido y se le da un relleno de 10px en la parte superior e inferior. Además, se utiliza margin-top: -8px; para ajustar el espacio superior. La clase .navegacion se aplica a una lista que se convierte en una fila flexible utilizando display: flex, y justify-content: space-around distribuye los elementos de manera uniforme.

.enlaces

.enlaces {

text-decoration: none;

border-bottom: 2px solid rgb(239, 91, 91);

padding: 0 10px 10px 10px;

color: rgb(198, 5, 5);

}

La clase .enlaces aplica un estilo a los enlaces eliminando el subrayado por defecto con text-decoration: none, y agrega un borde inferior de 2px con un color rojo. También se agrega un relleno en la parte inferior de 10px.

.principal

.principal {

margin: 40px 60px;

}

La clase .principal agrega márgenes de 40px en la parte superior e inferior, y 60px en los lados izquierdo y derecho del contenido.

img

img {

height: 200px;

}

El estilo aplicado a las imágenes establece una altura de 200px, lo que controla el tamaño de las imágenes en la página.

#lista_ordenada y #lista_desordenada

#lista_ordenada {

color: palevioletred;

}

#lista_desordenada {

color: lightskyblue;

}

Los estilos de #lista_ordenada y #lista_desordenada cambian el color del texto de las listas ordenadas a palevioletred y el color del texto de las listas desordenadas a lightskyblue.

.link

.link {

font-size: 20px;

}

La clase .link define el tamaño de la fuente de los enlaces en 20px.

.codigo

.codigo {

background-color: lightgrey;

padding: 10px;

width: auto;

font-size: 16px;

}

La clase .codigo se utiliza para dar un fondo gris claro a los bloques de código, añadir un relleno de 10px y establecer el tamaño de la fuente a 16px para mejorar la legibilidad del código.
