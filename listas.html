¿Cómo se crea una lista?

Hay dos tipos diferentes de listas:

Lista ordenada


    <ol>
        <li>Elemento 1</li>
        <li>Elemento 2</li>
        <li>Elemento 3</li>
    </ol>
  1. Elemento 1
  2. Elemento 2
  3. Elemento 3

Lista no ordenada


    <ul>
        <li>Elemento</li>
        <li>Elemento</li>
        <li>Elemento</li>
    </ul>
  • Elemento
  • Elemento
  • Elemento

La etiqueta se llama ol por Ordered List, ul por Unordered List y li viene de List Item.
